# Week 5 Functions

1. *f*(*x*) = *sin*(**k** &sdot; **x**)

*where* **k**, **x** are vectors \<k<sub>0</sub>, k<sub>1</sub>\>, \<x<sub>0</sub>, x<sub>1</sub>\>, and **k** is constant.

2. *f*(*x*) = e<sup>*i*__k__**x**</sup> = *cos*(**k** &sdot; **x**) + *isin*(**k** &sdot; **x**)

3. *f*(*x*) = *sin*(*kx* + *wt*)
